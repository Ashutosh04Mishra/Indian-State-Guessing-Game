<h1>Indian State Guessing Game</h1>

<h2>🛠️ Technologies Used</h2>

Python 🐍

Turtle Graphics

Pandas

CSV dataset for state coordinates

<h2>▶️ How to Run the Game</h2>

1. Clone the repository
  git clone https://github.com/Ashutosh04Mishra/Indian-State-Guessing-Game.git
  
2. Navigate to the project folder
3. Run main.py

<h2>🎯 How to Play</h2>
<ol>
A map of India will appear.

You will be prompted to guess the name of a state.

If the guess is correct:

The state name appears on the map.

Continue guessing until you find all states.
type "exit" to stop the game.
  
</ol>
